# VisualAlgo

An animated visualization of sorting algorithms.

<h2>Animated Algorithms</h2>
<h3>Available</h3>
<ul>
<li>Bubble Sort</li>
<li>Comb Sort</li>
<li>Insertion Sort</li>
<li>Selection Sort</li>
</ul>

<h3>Ongoing</h3>
<ul>
<li>Merge Sort</li>
<li>Shell Sort</li>
</ul>
